IMDb Rating API

Query IMDb movie ratings by ID

Endpoint: /api/rating

Method: GET

Parameter: imdbId - IMDb ID (tt + 7-8 digits)

Response Example:

{
  "imdbId": "tt12637874",
  "rating": 8.3,
  "numVotes": 366647,
  "updatedAt": "2026-01-25T13:50:30.998Z"
}

For personal use. Information courtesy of IMDb. Used with permission.

Rating data is not synchronized with the official website in real-time. Usually updated daily.
